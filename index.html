Langsung Jalan

Cukup siapkan Google Sheets kamu, dan sistem langsung bisa digunakan tanpa coding sama sekali.

1

Salin Template Sheets

Duplikasi template Google Sheets Doremi yang sudah disiapkan ke Google Drive kamu.

2

Pasang Script API

Tambahkan Google Apps Script yang sudah jadi — copy paste, deploy, salin URL-nya.

3

Isi Config Dashboard

Tempel URL API ke file config dashboard. Simpan, dan sistem sudah terhubung!

4

Kelola Sekolahmu

Login, tambah data siswa, catat SPP, kirim reminder, dan unduh laporan kapan saja.

Data Real-Time di Google Sheets Kamu

Semua data tersimpan langsung di Google Sheets yang kamu miliki. Tidak ada server yang harus dikelola, tidak ada biaya hosting tambahan.

  • Buka Sheets kapan saja untuk cek atau edit manual
  • Data otomatis sinkron dengan dashboard
  • Backup otomatis oleh Google Drive
  • Bagikan akses ke guru atau staf lain
  • Gratis selamanya selama punya akun Google
Pilihan Program

Tipe Pembayaran SPP

Sistem mendukung dua model pembayaran yang bisa disesuaikan per siswa.

Bulanan
Rp 350K/bln
Dibayar setiap tanggal 1–10
  • 4–5 pertemuan per bulan
  • Progress report mingguan
  • WhatsApp reminder H-3
  • Sertifikat akhir semester
HEMAT 15%
Triwulan
Rp 900K/3bln
Bayar sekali untuk 3 bulan
  • 12–15 pertemuan per kuartal
  • Progress report mingguan
  • Hemat Rp 150.000 vs bulanan
  • Evaluasi khusus akhir kuartal
  • Prioritas jadwal pilihan
