Integrasi Google Sheets
Hubungkan dashboard ini dengan Google Sheets sebagai database
⚡ Cara kerja integrasi Dashboard ini membaca & menulis data ke Google Sheets via Google Apps Script Web App. Semua data siswa, status bayar, dan progres mingguan tersimpan di Sheets. Pengingat email otomatis dikirim oleh Apps Script setiap pagi.
Langkah setup Google Sheets
1
Buat Google Spreadsheet baru
Buka sheets.google.com → Buat spreadsheet baru → beri nama SPP Sekolah Musik
2
Buat sheet dengan nama yang tepat
Buat 2 tab sheet: Siswa dan Log. Sheet Siswa untuk data utama, Log untuk riwayat pembayaran.
3
Buka Apps Script
Di Spreadsheet → menu Extensions → Apps Script. Hapus kode yang ada, paste kode dari file apps-script.gs yang disertakan.
4
Deploy sebagai Web App
Klik Deploy → New deployment → pilih type Web app → Execute as: Me → Who has access: Anyone → klik Deploy.
5
Copy URL Web App
Setelah deploy, copy URL yang muncul. Paste di kolom bawah ini lalu klik Simpan.
6
Set trigger harian untuk pengingat email
Di Apps Script → Triggers (ikon jam) → Add Trigger → pilih fungsi kirimPengingat → Time-driven → Day timer → jam 07:00.
Konfigurasi URL Web App
Setelah URL disimpan, data siswa akan otomatis disinkronkan ke Google Sheets.
